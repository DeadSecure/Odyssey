# Odyssey

A unified monitoring service built on X-ray Core to fully monitor X-ray based services.


# Installation 

1. Clone the repository

```bash
git clone https://github.com/Javadyakuza/Odyssey.git
```

2. Install dependencies
```bash
npm install
```

3. Run the Wizard 
```bash
npm run start
```
